Known issues:

  • Animations are lagging in Internet Explorer 9
  • Keyboard/Input Quiz can not be used with current version of Chrome iOs(iPad). It might be possible to correct.
  • Letters sometimes remain in position in Jumbled/Scrambled quiz on Safari, iPad. Use Chrome iOs until fix/rewrite.
  • IOS 8 issues:
    • iframe displacement on input touch
    • prevent ghost clicks on close event
  • Safari browser 8.2 Ipad, minor issue: If the ipad is locked/covered while displaying Safari and then subsequently unlocked, the animations will run less smoothly. Lag goes away if another browser tab is opened/viewed, or if Safari is restarted.
  • Minor issue: Occasionally, the drop function does not fire in transition quiz, and the container will exceed the maximum allowed height.
